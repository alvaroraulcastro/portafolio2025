# Portafolio — Álvaro Raúl Castro

Este portafolio fue creado con Next.js, React y TailwindCSS para presentar conocimientos y experiencia profesional.

## Estado

Realizado el 29 de Abril de 2025.

## Requisitos

- Node.js 18+ (recomendado)
- npm o pnpm

## Instalación (local)

1. Instalar dependencias:

```bash
npm install
```

2. Ejecutar en desarrollo:

```bash
npm run dev
```

3. Construir para producción:

```bash
npm run build
npm run start
```

## Scripts útiles

- `npm run dev` — servidor de desarrollo (Next.js)
- `npm run build` — compilar para producción
- `npm run start` — iniciar la app compilada
- `npm run lint` — ejecutar linter (Next.js)
- `npm run typecheck` — comprobación TypeScript
- `npm run format` — formatear el código con Prettier (añadir `prettier` devDependency)

> Nota: si agrego dependencias dev como `prettier` o `eslint`, ejecuta `npm install` después de actualizar `package.json`.
